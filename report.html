Milestone 2 Report

  1. github repo url: https://github.com/GinnySong/BookEmporium
  2. url to the deployed application: https://ginnysong.github.io/BookEmporium/index.html
  3. this is report.html, tada!
  4. media queries:
    • books.css: books are displayed in rows of 3 on large screens and 1x1 on small screens
    • forms.css: forms elements are displayed as inline blocks (horizontally) on large screens and as blocks (vertically) on small screens
    • shared.css: the navbar nav-links are collapsed on small screens and expanded on large screens (using bootstrap)
  5. Five separate pages:
    • index.html
    • search.html
    • collection.html
    • wishlist.html (pending)
    • readlater.html (pending)
  6. header: the navbar serves as the header. it includes navigation links and the website logo (aka the website title)
  7. nav-links: located in the navbar! (home, search, collection, to-read (pending)
  8. images: currently the only images included in the site are the book covers, which are accessed via url and not stored locally. the alt attributes of each of the book covers are the book title followed by the author.
  9. validation: every html page and stylesheet validates without warnings or errors using the Nu Validator
  10. font faces:
    • heading fonts are rendered in the following order: 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif
    • paragraph fonts are rendered in the following order: Georgia, 'Times New Roman', Times, serif
  11. form: the site includes a search form. The input is required, so the url sent to the API cannot be empty.
  12. links to external sites: the book information modal will include a link to a wikipedia article about the book.
  13. network requests: calling Open Library book API. may call amazon to pull up a listing for each book.
